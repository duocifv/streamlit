## Chain of Thoughts

### 1. Analysis & Planning

- **Project Goal:** Tạo landing page sản phẩm Cà Phê Sáng Tạo, hiện đại, responsive, minimalism + micro-interactions + glassmorphism.
- **Target Audience:** Người yêu cà phê, khách hàng online, khách hàng thân thiết.
- **Tech Stack:** HTML + CSS + JS trong 1 file duy nhất. Không dùng thư viện ngoài.
- **UX/UI Considerations:** Responsive & Fluid Layout: Nội dung tự co giãn theo mọi kích thước màn hình (desktop, tablet, mobile). Flexible Grid: Dễ mở rộng, tự động cân chỉnh spacing và card, hiệu ứng hover, smooth scrolling, back-to-top button, card Glassmorphism cho Features, Services, Testimonial.
- **Icons:** Mỗi mục (Features, Services, CTA, v.v.) có thể kèm icon, dùng tên/mã từ https://iconify.design/ để hiển thị trên web.

### 2. Frontend Development

- **Header:** Menu có anchor link đến các section. Logo/brand name nổi bật.
- **Hero Section:** Hình ảnh sản phẩm (dùng [PRODUCT_IMAGE]), mô tả ngắn, nút call-to-action.
- **Product Intro:** Giới thiệu sản phẩm, mô tả chi tiết.
- **Features Section:** 3 tính năng nổi bật, card Glassmorphism, hover effect.
- **Gallery Section:** Hình ảnh sản phẩm bổ sung (Picsum).
- **Services Section:** 3 dịch vụ chính, card Glassmorphism.
- **Testimonial Section:** 2-3 feedback của khách hàng, card Glassmorphism, slider nhỏ nếu cần.
- **FAQ Section:** 2-3 câu hỏi thường gặp, accordion Micro-interactions.
- **Call-to-action Section:** Nút “Mua Ngay” hoặc “Liên hệ”, hiệu ứng hover Glassmorphism.
- **Footer:** Thông tin liên hệ, bản quyền, back-to-top button.
- **Micro-interactions:** Hover, scroll animation, button effect.

### 3. Deployment & Documentation

- **Output:** 1 file HTML chứa tất cả: HTML + CSS + JS.
- **Responsive:** Desktop và Mobile.
- **Color Theme:** [PRIMARY_COLOR] áp dụng thống nhất.
- **Image Placeholder:** Sử dụng [PRODUCT_IMAGE] hoặc Picsum.photos.
